# 성인 수동 입력 자료

백오피스(`/kcc/saints`)에 **수동으로 추가/수정할 성인 항목**을 정리해 두는 문서입니다.
maria.catholic.or.kr 자동 임포트에 없거나 보완이 필요한 성인을 여기에 먼저 정리한 뒤,
백오피스 폼에 옮겨 넣습니다. (수동 저장 시 `source='manual'`이 되어 재임포트해도 덮어써지지 않습니다.)

## 현재 스키마로 담을 수 있는 필드

`saints` 테이블/백오피스 폼은 **기본 메타데이터만** 보관합니다.

| 폼 필드 | 설명 |
|---------|------|
| `sourceSaintId` | 기본키(정수, 필수). 수동 항목은 import ID와 겹치지 않는 큰 정수 사용 |
| `nameKo` | 한글명(필수) |
| `nameLatin` | 라틴/영문명 |
| `feastMonth` / `feastDay` | 축일 월(1-12) / 일(1-31), 비워도 됨 |
| `status` | 신분(예: 순교자) |
| `kind` | 등급(성인 / 복자 등) |
| `regionKo` / `regionEn` | 지역(한글 / 영문) |
| `yearText` | 연도 원문(예: `+34년경`) |
| `detailUrl` / `url` | 상세/원문 링크(선택) |

> ⚠️ **설명 본문(약력/묵상 등 긴 텍스트)을 담는 필드는 아직 없습니다.**
> 아래 각 항목의 "설명 본문"을 실제로 저장하려면 `saints`에 `description text` 컬럼과
> 백오피스 textarea 입력란을 추가해야 합니다(추후 결정). 또한 앱은 아직 성인 데이터를
> 화면에 노출하지 않으므로, 저장하더라도 앱에는 당장 보이지 않습니다.

## 스테파니아 (12월 26일)

내 아내 축일. 주보성인은 **스테파노 첫 순교자**.

### 메타데이터 (지금 바로 입력 가능)

| 필드 | 값 |
|------|-----|
| sourceSaintId | `9991226` (수동용 임의 정수 - import ID와 겹치지 않으면 됨) |
| nameKo | `스테파니아` |
| nameLatin | `Stephania` |
| feastMonth | `12` |
| feastDay | `26` |
| status | `순교자` |
| kind | `성인` |
| regionKo | `예루살렘` (선택) |
| regionEn | `Jerusalem` (선택) |
| yearText | `+34년경` (스테파노 성인 순교, 약 34년) |
| detailUrl / url | (선택) |

- `스테파니아`는 `스테파노`의 여성형 세례명이라 status/kind는 주보성인(스테파노 첫 순교자) 기준으로 채웠습니다.
- 원하면 별도로 `스테파노` 항목을 하나 더 만들어 검색에서 둘 다 잡히게 할 수 있습니다.

### 설명 본문 (description 필드가 생기면 사용)

**① 첫 부제**

'스테파니아'는 '스테파노'의 여성형 이름으로, 스테파노 성인을 주보성인으로 모시는 세례명입니다.
스테파노 성인은 열두 사도에게 안수받은 교회 최초의 부제 일곱 명 중 한 사람입니다. 은총과 능력이 충만해 큰 이적과 표징들을 일으켰습니다.

> "그의 말에서 드러나는 지혜와 성령에 대항할 수가 없었다." - 사도 6,10

**② 첫 순교자**

당시 사람들은 스테파노 성인이 모세와 하느님을 모독했다고 모함했습니다. 그러나 성인은 성령이 충만해 지혜롭게 반박했고, 화를 이기지 못한 사람들이 끝내 돌을 던져 죽였습니다.
이로써 스테파노 성인은 교회 역사상 첫 순교자로 기록되었습니다.

**③ 은총과 능력이 충만한 성인**

'스테파노', '스테파니아'는 그리스어 동사 στεφανόω(stephanou, 스테파노우)에서 유래한 이름으로, 탁월한 업적에 상을 주거나 머리에 꽃 장식을 씌워준다는 의미입니다.
성인은 그 이름처럼 뛰어난 언변과 지혜로 그리스도를 열정적으로 증언했습니다.

**④ 믿음을 실천한 성인**

스테파노 성인은 살아있는 동안 열정을 다해 그리스도를 증언했고, 순교하는 순간에도 원수들을 위해 기도했습니다.

> "주님, 이 죄를 저 사람들에게 돌리지 마십시오." - 사도 7,60

믿음과 성령의 이끄심으로 어디서나 굽힘 없이 하느님을 전하고 마침내 하늘나라의 영광을 얻었습니다. 스테파노 성인을 주보로 모신다면 그 이름처럼 당당하고 훌륭한 신앙인이 되지 않을까요?
