# 사용자 확인·작업 목록 (ACTION ITEMS)

한국 가톨릭 전례력 앱을 완성/배포하기 위해 **당신이 직접 검토·결정·삽입해야 할 항목**을 정리했습니다.
우선순위 높은 순으로 정렬했습니다. (⚠️ = 정확성·법무상 반드시 확인)

---

## 1. ⚠️ 전례력 데이터 검토·보완 (가장 중요)

데이터는 코드가 아니라 **편집 가능한 JSON 파일**에 있습니다:
`assets/calendar/general.json` (일반 로마력), `assets/calendar/korea.json` (한국 고유), `assets/calendar/adaptation.json` (국가 조정 정책).
편집 후 앱을 다시 실행/빌드하면 반영됩니다. (형식은 각 파일의 기존 항목을 그대로 따라 하면 됩니다.)

### 1-1. ⚠️ 한국 고유 전례력 (`korea.json`) — CBCK 공식 전례력으로 검증 필수
- [ ] **성 김대건 안드레아 사제와 성 정하상 바오로와 동료 순교자 (9월 20일)** 등급이 **대축일**이 맞는지 확인. (현재 대축일=solemnity로 설정)
- [ ] **복자 윤지충 바오로와 동료 순교자 (5월 29일)** 날짜·등급·공식 명칭 확인. (현재 의무 기념일=obligatoryMemorial)
- [ ] 그 밖의 한국 고유 축일·기념일 **추가 수록**: 예) 최양업 토마스 신부, 절두산 순교자, 주교회의가 지정한 기타 기념일 등.
- [ ] "한국 교회의 수호자"(원죄 없이 잉태되신 마리아)·"한국 성직자들의 수호자" 등 특별 지정이 전례력 등급/명칭에 반영돼야 하는지 확인.

### 1-2. 일반 로마력 (`general.json`) — 선택 기념일 로스터 보완
- 현재 **모든 대축일·축일 + 주요 의무 기념일**은 수록되어 있으나, **다수의 선택 기념일(성인 기념일)이 미수록**입니다.
- [ ] 로마 미사 경본/주교회의 전례력 기준으로 나머지 성인 기념일을 `general.json`에 추가. (색·등급 필드 참고: rank는 `optionalMemorial`, color는 성인 유형에 따라 white/red)
- [ ] 수록된 축일의 **공식 한국어 명칭**이 주교회의 표기와 일치하는지 검토. (현재는 일반적 표기 기준)

### 1-3. 의무 축일 목록 (`adaptation.json` → `holyDaysOfObligation`)
- [ ] ⚠️ 한국의 의무 축일이 **주일 + 주님 성탄(12/25) + 천주의 성모 마리아(1/1)** 외에 더 있는지 CBCK 확인 후 배열에 id 추가/삭제.

> ✅ 이미 CBCK 2026 전례력으로 검증 완료: 주님 공현→주일(1/4), 주님 승천→주일(5/17), 성체 성혈→주일(6/7) 이동. 부활·재의수요일·그리스도왕 등 이동 축일 날짜 계산 일치.

## 2. 제품·배포 결정 (당신이 정해야 함)

- [ ] **앱 이름·아이콘·브랜딩**: 현재 "가톨릭 전례력". 앱 아이콘은 Flutter 기본 이미지 → **교체 필요**.
- [ ] **번들 ID**: 현재 임시로 `app.catholiccalendar` (iOS `app.catholiccalendar.catholicCalendar`). 스토어 등록 전 **조기에 확정** (나중에 변경 어려움).
- [ ] **웹 호스팅 선택**: Cloudflare Pages / Firebase Hosting / GitHub Pages 중 택1 + 도메인. (SPA fallback 설정 필요 — `/2026/07` 같은 URL 때문)
- [ ] **주 시작 요일**: 현재 일요일 고정. 월요일 시작 옵션이 필요하면 알려주세요.
- [ ] **표시 범위 확정**: 현재 전례시기·전례색·축일/기념일·주일/전례일 명칭·독서주기(가나다해/ⅠⅡ)를 모두 표시. (독서 본문은 제외 — 확정됨)

## 6. 검증 방법 (직접 확인)

```bash
# 엔진 단위/골든 테스트 (110개)
dart test packages/liturgical_calendar

# 앱 위젯 테스트
flutter test

# 웹에서 실제 확인 (브라우저 열림)
flutter run -d chrome
# → 임의의 월을 열어 전례색·명칭·상세가 CBCK 전례력과 일치하는지 육안 대조 권장

# 웹 정적 빌드
flutter build web --release   # 산출물: build/web/
```

- ⚠️ **출시 전 반드시** 2025~2027년 CBCK 공식 전례력과 월별로 대조해 축일·등급·전례색을 확인하세요. 특히 한국 고유 축일(1-1)과 선택 기념일(1-2)이 미완성입니다.
